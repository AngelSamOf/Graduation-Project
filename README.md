## Основная задумка

Игрок отыгрывает роль древних духов мира, которые стараются защитить вселенную от загадочного заражения. 

У многих планет появились заражённые кометы, которые распространяют порчу по мирам. Когда порча порабощает мир, он полностью умирает.

Спасите планеты от уничтожения, используя уникальные способности духов.
